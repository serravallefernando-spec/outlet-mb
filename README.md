# Gestionale Outlet — Maurizio Boutique

App React per la gestione del magazzino e delle vendite dell'outlet.

## Deploy su Cloudflare Pages

1. Carica questa cartella su GitHub come nuovo repository
2. Vai su [Cloudflare Pages](https://pages.cloudflare.com)
3. Crea un nuovo progetto → collega il repo GitHub
4. Impostazioni di build:
   - **Framework preset:** Vite
   - **Build command:** `npm run build`
   - **Output directory:** `dist`
5. Salva e deploya

## Sviluppo locale

```bash
npm install
npm run dev
```
